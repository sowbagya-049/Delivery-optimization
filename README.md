# Delivery Optimization System

## Overview
This project solves a logistics delivery optimization problem using multiple algorithms and decision-making techniques.

## Methods Used
1. Greedy Load Balancing with Priority Weighting
2. K-Means Clustering

## Decision Technique
A Multi-Criteria Decision Making (MCDM) approach was used based on:
- Variance (Load Balance)
- Maximum Distance (Makespan)
- Priority Score

## Features
- Handles multiple input files
- Edge case testing
- Automatic output generation
- Visualization using plots

## Output
- Optimized delivery plan
- Comparison plots
- Final method selection
- Summary report
